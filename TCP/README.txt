To run the program
1. Run the server via command prompt by inputting: python .\EnhancedTCPserver.py
2. (localhost) Run the client via different command prompt by inputting: python .\EnhancedTCPclient.py localhost 9999
2. (ip address) Run the client via different command prompt by inputting: python .\EnhancedTCPclient.py ip address 9999
3. Follow the prompts outputted by the client.
